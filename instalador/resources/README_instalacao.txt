FRP Cleaner Samsung — Guia Rápido
=================================

Conteúdo do instalador:
- FRP-Cleaner.exe (executável PyInstaller)
- Documentação básica de uso

Requisitos pós-instalação:
1. Windows 10/11 com WSL 2 ativo e distribuição Ubuntu/Debian instalada.
2. Utilitários externos: simg2img, android-tools-fsutils, avbtool, magiskboot, fuse (instalados dentro do WSL).
3. Pastas necessárias para a execução: imagem system.img, pasta AP com ficheiros da ROM e pasta Keys com avb_private.pem e magiskboot.

Passos após a instalação:
1. Abra o atalho "FRP Cleaner Samsung".
2. Selecione a imagem system.img (ou sparse.img), a pasta AP e a pasta Keys.
3. Clique em "Executar Limpeza" (GUI) ou utilize o CLI conforme descrito no README principal.

Para suporte detalhado consulte o ficheiro README.md na raiz do projecto ou visite o repositório oficial.
